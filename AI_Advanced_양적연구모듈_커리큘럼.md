## Markdown 파일로 산출물을 관리하는 이유

### 왜 `.md` 파일인가?

본 과정의 모든 산출물(연구 설계서, 통계 해석 보고서, Figure 설명, 최종 보고서)은 Markdown(`.md`) 형식으로 작성한다. 그 이유는 다음과 같다.

| 이유 | 설명 |
|------|------|
| **GitHub 공유 최적화** | `.gitignore`로 PDF·PPT 등 대용량 바이너리를 제외하고 텍스트 기반 `.md` 파일만 공유 → 저장소 용량 최소화, 변경 이력 정확한 추적 |
| **버전 관리** | 매 회차 커밋(commit)으로 연구 진행 과정의 이력이 남아 수정 전후 비교 가능 |
| **RAG 입력 데이터** | AI에게 연구 맥락을 주입할 때 `.md` 파일을 NotebookLM·TimelyGPT·Claude에 직접 업로드하여 RAG(Retrieval-Augmented Generation) 기반 정확한 응답 유도 |
| **플랫폼 이관성** | Markdown은 특정 플랫폼에 종속되지 않으므로 Notion·Obsidian·VS Code 등 어디서나 열람·편집 가능 |
| **AI 생성 → 문서 연결** | AI가 생성한 텍스트를 그대로 `.md`로 저장하면 추가 변환 없이 바로 GitHub에 업로드 |

### RAG(Retrieval-Augmented Generation) 활용 전략

RAG는 AI가 사전 학습 데이터만이 아닌, **사용자가 직접 제공한 문서를 검색·참조하여 답변을 생성**하는 방식이다. 본 과정에서는 다음과 같이 응용한다.

| 단계 | MD 파일 역할 | 활용 도구 |
|------|------------|---------|
| **연구 설계 → 설문 문항** | 연구 설계서 `.md`를 업로드 → "이 설계에 맞는 설문 문항을 Likert 척도로 생성해" | NotebookLM, TimelyGPT RAG |
| **통계 결과 → 해석 텍스트** | 통계 결과 표를 `.md`로 저장 → "이 표에서 유의미한 결과를 APA 형식으로 해석해" | Gemini, Claude |
| **Figure → Caption** | Figure 설명 `.md`를 업로드 → "이 Figure에 맞는 학술 Caption을 생성해" | Claude, ChatGPT |
| **전 회차 보고서 → 결론 생성** | 5~7회차 산출물 `.md` 전체 업로드 → "이 자료들을 종합해 연구 결론을 작성해" | NotebookLM (소스 통합), Gemini |
| **프로토콜 → 재사용** | 완성된 AI 활용 프로토콜 `.md`를 Gems 소스로 등록 → 이후 연구에서 동일 워크플로우 재사용 | Google Gems |

> **실습 흐름 요약:** AI가 생성한 텍스트 → `.md` 저장 → GitHub 커밋 → 다음 회차에서 해당 `.md`를 AI에 업로드(RAG) → 더 정교한 후속 결과 생성 → 다시 `.md` 저장 → 반복

## 프롬프트 예시 10선

양적연구 트랙 5~8회차에서 바로 사용할 수 있는 실전 프롬프트.

### ① 연구 문제·가설·변수구조 생성
```
당신은 양적 연구 설계 전문가입니다.
아래 연구 주제를 바탕으로 다음 항목을 작성하십시오.

연구 주제: [주제 입력]
연구 배경: [2~3문장 설명]

작성 항목:
1. 연구 문제 (Research Question) 3개
2. 각 연구 문제에 대응하는 연구 가설 (H1, H2, H3)
3. 독립변수·종속변수·제어변수 목록 (변수명, 측정 수준, 자료 유형 포함)
4. 적합한 연구 설계 유형 (횡단/종단/실험/준실험) 및 선택 이유
```

### ② 설문 문항 자동 생성 (Likert + RAG)
```
아래 연구 설계 문서를 참고하여 [변수명]을 측정하기 위한 설문 문항을 생성하십시오.

[연구 설계 문서 내용]: (연구 설계 .md 파일 내용을 여기에 붙여넣기)

설문 문항 조건:
- Likert 5점 척도 (1=전혀 그렇지 않다 ~ 5=매우 그렇다)
- 문항 수: 5개 (역채점 문항 1개 포함)
- 긍정적·중립적 언어로 작성
- 각 문항 번호, 문항 내용, 역채점 여부를 마크다운 표로 제시
```

### ③ 통계 결과 → APA 학술 해석 텍스트
